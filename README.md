# VideoSlicer

Free macOS video frame extractor. Extract first and last frames from any video as full-resolution PNGs.

## What is VideoSlicer?

VideoSlicer is a free, native macOS app that extracts the first and last frames from any video file and saves them as lossless PNG images. Built with Apple's AVFoundation — no ffmpeg, no Homebrew, no command-line tools needed. Just drag and drop.

Perfect for AI video generation workflows with tools like Sora, Kling, Wan, and Veo.

## Features

- **Drag & Drop** — Drop any video file to extract frames instantly
- **First & Last Frame** — Extracts both opening and closing frames in one operation
- **Full Resolution PNG** — Lossless output at the video's native resolution
- **AI Video Ready** — Extract start/end keyframes for Sora, Kling, Wan, Veo, and other AI video tools
- **No Dependencies** — No ffmpeg, no Homebrew, no terminal commands
- **Multiple Formats** — MP4, MOV, AVI, M2V, and all macOS-native video formats
- **100% Offline** — All processing happens locally on your Mac
- **Multilingual** — Available in 8 languages
- **Universal Binary** — Apple Silicon and Intel
- **100% Free** — No ads, no subscriptions, no in-app purchases
